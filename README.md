# 🚗 Quikr Car Data Analysis

## 📌 Project Overview

This project focuses on cleaning, processing, and analyzing used car data from Quikr to extract meaningful insights about pricing and trends.

---

## 🧠 Key Features

* Data cleaning (handling missing values, inconsistent formats)
* Exploratory Data Analysis (EDA) using Python
* Visualization of key relationships and patterns

---

## 📊 Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook

---

## 📈 Key Insights

* Newer cars tend to have higher prices
* Cars with lower kilometers driven are more expensive
* Fuel type has an impact on pricing
* Certain brands dominate the used car market

---

## 🎯 Outcome

This project demonstrates data cleaning, visualization, and analytical skills by uncovering key factors affecting car prices.
